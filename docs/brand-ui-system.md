# Brand / UI System

## 品牌气质
- 名称：Knowledge Studio
- 感受：清醒、克制、专业、偏创作者工具感
- 核心承诺：帮你把知识内容的“前置思考”做快、做稳

## 视觉系统
- 深色背景：表达“工作台”而不是“营销页”
- 蓝色主色：理性、可信
- 青绿色辅助色：表达生成与结构化

## 组件
- Hero card：当前 brief 摘要
- Topic ranking：快速判断优先选题
- Form card：输入 brief
- Result card：输出结构 / prompt / 清单
- Angle card：多个传播角度

## 交互原则
1. 一进来就能操作
2. 每一步都在减少创作者的不确定性
3. 输出必须一眼可复制和继续加工

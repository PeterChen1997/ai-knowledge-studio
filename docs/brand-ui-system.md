# Knowledge Studio Brand / UX System

## 产品气质重构
参考 SnapRead 的品牌方法，Knowledge Studio 的气质不应该是“炫 AI”，而应该是：

- Quiet intelligence
- Research-driven
- Signal over noise
- Structured thinking
- Productive reflection

这意味着：
- 不把 AI 表现成魔法
- 不让页面像营销页
- 不用太多“立即生成爆款内容”的语言
- 更强调：筛选、提炼、对照、反驳、沉淀

---

## 品牌关键词
- Signal
- Frictionless triage
- Counterview
- Perspective
- Structured creation
- Quiet studio

---

## UI Naming 建议
借鉴 SnapRead 用工作流命名而非功能命名的方式：

- **Dashboard**：今天的阅读与创作总览
- **Inbox**：材料输入池
- **Reader**：阅读与思辨面板
- **Projects**：内容项目工作区
- **Library**：模板、角度、反驳框架、资料库
- **Studio**：AI 提供商与系统设置

避免用：
- “Prompt 生成页”
- “内容包页”
- “AI 页面”

---

## 视觉原则
1. **像工作台，不像官网**
2. **像阅读分析器，不像内容营销工具**
3. **信息密度高，但节奏平静**
4. **重点是“判断”和“对照”，不是“炫生成”**

---

## Reader 页面原则
Reader 是未来最重要的页面，应该具备：

### 左侧
- 原文 / 摘录 / 来源信息
- AI Summary

### 右侧
- Core Claims
- Counterview
- Rebuttals
- Open Questions
- Content Opportunities

### 底部动作
- Save angle to project
- Save rebuttal framework
- Create project from article

这会让产品真正从“读懂”走向“形成自己的判断”。

---

## 语气建议
多用：
- 筛选
- 提炼
- 角度
- 反驳
- 问题
- 判断
- 沉淀

少用：
- 爆款
- 一键出稿
- 自动写完
- 全自动内容工厂

## 一句话品牌表达
### 中文
Knowledge Studio 是一套帮助知识创作者筛选值得深入的内容、展开正反思考，并把判断沉淀成创作项目的工作台。

### 英文
Knowledge Studio is a research-to-content workspace that helps creators filter what deserves depth, think through multiple perspectives, and turn judgment into reusable creative assets.
